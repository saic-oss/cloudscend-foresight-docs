# Frequently Asked Questions

## General

1. How often should I analyze my application portfolio using the Sage product?

   - It is recommended to run Sage tools against your portfolio at least every quarter in order to see how your portfolio is trending over time.

2. How long does it take for Sage to analyze an application?

   - Depending on the size of the application, it usually takes Sage less than 5 min to run analysis on any application.

3. What browsers are supported by Sage?

   - Sage is compatible with all versions of Chrome, Microsoft Edge, Firefox ESR or higher, and Safari 5.1 or higher.

4. Is there a limit to the number of users I can have with my Sage account?

   - No, your Sage administrator can add as many users as you would like to your Sage instance.

5. How many applications can I onboard?

   - When you purchase Sage, you receive a license for the number of applications you selected. If you use all your licenses, you can always purchase more at **LINK TO PURCHASE SITE**

6. Does Sage interface with source code configuration management tools?

   - No, Sage does not interface with any source code configuration management tools. You must extract your source code from your configuration management tool and compress it into a zip file before uploading it to Sage.
